# 今年我当爸爸了 👶

今年我当爸爸了。

家里迎来了一个马年男宝宝。看到他的那一刻，我才真正意识到，人生真的进入了一个全新的阶段 :) 

这一年最辛苦的人，其实是妈妈。怀孕不容易，生孩子更不容易。看着老婆经历生产，才知道那些在旁人看来轻描淡写的几个字，背后到底要承受多少疼痛、紧张和消耗。她真的很了不起。❤️

从月子中心出来回到家以后，生活一下子变得兵荒马乱。以前总觉得带娃这件事，辛苦归辛苦，应该总能慢慢适应；真正轮到自己，才发现每天都有新的状况。会担心他吐奶，担心是不是拍嗝没拍好；会盯着尿布发呆，担心排便次数太少，是不是哪里不舒服。很多时候明明已经很累了，脑子里却还是停不下来。>_<

带娃真的很辛苦，睡眠被切得零零碎碎，时间也完全不再属于自己。但也是在这些手忙脚乱的时刻里，我慢慢感受到另一种很具体的幸福。和老婆一起半夜起来照顾宝宝，一起研究他的每一点小反应，一起在疲惫里互相搭把手，虽然累，却也觉得这个小家变得特别真实、特别柔软。🥹

现在回头看，最想说的还是感谢老婆。谢谢你拼尽全力把宝宝带到这个世界，也谢谢你在最累的时候，依然把这个家撑了起来。以后我们就一起继续慢慢学，学着怎么把孩子带好，学着怎么成为更好的爸爸妈妈。辛苦了，老婆呀 (｡･ω･｡)

希望小家伙平安健康地长大，也希望我们一家三口，在热闹又琐碎的日子里，慢慢积攒属于自己的幸福。☀️
